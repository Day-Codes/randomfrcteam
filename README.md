## Run Locally

Clone the project

```bash
  git clone https://github.com/Day-Codes/randomfrcteam.git
```

Go to the project directory

```bash
  cd randomfrcteam
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run start
```
